# togather

각종 모임과 같은 취미를 가진 사람들을 이어주는 웹 어플리케이션 서비스입니다.

## 개발 환경

* Intellij IDEA Ultimate 2022.1.3
* Java 11
* Gradle 7.4.1
* Spring Boot 2.7.2

## 기술 세부 스택

Spring Boot

* Spring Web
* Spring Data JPA
* Thymeleaf
* Spring Security
* OAuth2
* Validation
* H2 Database
* MySQL Driver
* Lombok
* Spring Boot DevTools
* Spring Configuration Processor

그외 

* QueryDSL 5.0.0
* Bootstrap 5.2.0-Beta1
